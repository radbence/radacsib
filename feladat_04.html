4. Feladat
Írj egy programot, amely bekéri a nevedet, és kiírja a páros számokat 1 és 10 között! (2 4 6 8 10)

A program forrása:

        
for szam in range(1, 11):
    if szam % 2 == 0:
        print(f"{szam}", end = ' ')
print()
print(f"A program vége")

A program futtatása >>>

for szam in range(1, 11): if szam % 2 == 0: print(f"{szam}", end = ' ') print() print(f"A program vége")
