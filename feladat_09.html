9. Feladat
Kérjünk be két egész számot, szám1 és szám2. Hasonlítsuk össze a két számot, és írassuk ki, hogy a szám1 kisebb mint a szám2, vagy a szám2 kisebb mint a szám1, vagy a szám1 egyenlő szám2-vel.

A program forrása:

        
szam1 = int(input("Szám 1: "))
szam2 = int(input("Szám 2: "))

if szam1 < szam2:
    print(f"{szam1} kissebb mint {szam2}")
elif szam1 == szam2:
    print(f"{szam1} ugyanannyi mint {szam2}")
elif szam1 > szam2:
    print(f"{szam1} nagyobb mint {szam2}")

A program futtatása >>>

szam1 = int(input("Szám 1: ")) szam2 = int(input("Szám 2: ")) if szam1 < szam2: print(f"{szam1} kissebb mint {szam2}") elif szam1 == szam2: print(f"{szam1} ugyanannyi mint {szam2}") elif szam1 > szam2: print(f"{szam1} nagyobb mint {szam2}")
