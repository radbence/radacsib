7. Feladat
Kérjünk be a billentyűzetről két egész számot és írassuk ki a két szám összegét a képernyőre!

A program forrása:

        
szam_1 = input("Kérek egy számot: ")
szam_1 = int(szam_1)
szam_2 = int(input("Kérek egy másik számot: "))
összeg = szam_1 + szam_2
print(f"A megadott két szám összege: {szam_1 + szam_2}")
print(f"A megadott két szám összege: {összeg}")

A program futtatása >>>

szam_1 = input("Kérek egy számot: ") szam_1 = int(szam_1) szam_2 = int(input("Kérek egy másik számot: ")) összeg = szam_1 + szam_2 print(f"A megadott két szám összege: {szam_1 + szam_2}") print(f"A megadott két szám összege: {összeg}")
